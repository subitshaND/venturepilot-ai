# VenturePilot AI

An AI-powered startup blueprint generator built using Flask and IBM watsonx AI (Granite).

 **Live Demo:** https://venturepilot-ai.onrender.com

---

## Project Overview

VenturePilot AI helps entrepreneurs transform startup ideas into professional business blueprints using IBM Granite Foundation Models.

The application generates:

- Executive Summary
- Business Model Canvas
- Competitor Analysis
- SWOT Analysis
- Budget Estimation
- Funding Recommendations
- Revenue Model
- Go-To-Market Strategy
- Six-Month Roadmap
- Future Scope
